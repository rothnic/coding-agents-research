# Coding Agents Research

A clean research workspace for studying coding-agent systems and distilling them into practical, minimal, local-first prototypes.

## Purpose

- Collect repeatable analyses of coding-agent repos/specs.
- Run isolated checkouts and branch-based experiments.
- Distill findings into reusable guides and build-ready specs.
